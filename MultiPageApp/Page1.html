MultiPageApp

Welcome to the MultiPageApp! This app is an example of creating a WebApp with multiple pages using pseudo-routing in Google Apps Script. Learn how to structure your app, manage dynamic content, and implement navigation seamlessly. This project is a foundation for building robust multipage WebApps with minimal effort!

The structure of this project is inspired from BPWEBS example Multiple Page Webapp Example , where you can find the original example having the foundations of a multiple page WebApp.
